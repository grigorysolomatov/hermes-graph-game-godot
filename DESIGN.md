# Graph Game — Design Document

## Core Concept
A node-and-edge resource simulation. Players place nodes on a map, connect them with directed edges, and watch resources flow between them each game tick.

---

## World & Time

- The world is a free-form 2D map with no grid.
- Time is measured in **in-game hours** (ticks).
- **1 working day = 8 ticks.**
- Speed settings:
  - **Pause** — no ticks occur.
  - **1×** — 1 tick per real second.
  - **2×** — 1 tick per 0.5 real seconds.

---

## Nodes

### General Rules
- Nodes are placed freely on the map by dragging a ghost copy from the node selection menu.
- Nodes can be moved by dragging them directly on the map.
- Tapping a node opens a **context menu** — a row of buttons appearing directly above the node.
- Default context menu buttons available on every node:
  - **Delete** — removes the node (and all its edges).
  - **Connect** — designates this node as an edge source; the next node tapped becomes the target.
- Each node has an **inventory** displayed beneath it: resource icon followed by count (e.g. 🤚 2).

### Worker
- **Color:** warm amber
- **Icon:** hand emoji
- **Produces:** 1 labor per tick.
- **Capacity:** 1 labor maximum. If already holding 1 labor, stops producing until the labor is moved away.

### Sea
- **Color:** muted teal/blue
- **Icon:** fish emoji
- **Converts:** all labor currently held → fish, once per tick (entire amount, not one by one).
- **Fish capacity:** unlimited.
- **Labor capacity:** unlimited (but all labor is converted each tick so it does not accumulate).

---

## Resources

| Resource | Icon | Notes |
|----------|------|-------|
| Labor | hand emoji | Generated by Worker; max 1 held by Worker |
| Fish | fish emoji | Generated by Sea from labor; no cap |

- All resource icons are emoji rendered as PNGs on a transparent background.
- In the inventory display, icons appear without a surrounding circle — just the emoji.

---

## Edges

- Edges are **directed** (source → target), displayed as arrows.
- Created via the Connect button on the source node, then tapping the target node.
- Each edge has a **resource type** assigned to it.
- Each tick, the edge moves **1 unit** of its assigned resource from source to target (if available).
- Tapping an edge opens a resource-type picker showing all available resource types globally.

---

## UI Layout

### Map
- The main area. Nodes and edges live here.
- Tapping empty space deselects any open context menus.

### Bottom Panel — Node Selection Menu
- Collapsible panel at the bottom of the screen.
- Contains available node types; drag one onto the map to place it.

### Right Panel — Time Controls
- Collapsible panel on the right side of the screen.
- Contains: Pause, 1×, 2× buttons.

---

## Visual Style
- Dark theme throughout.
- Node visuals: colored circle with emoji icon centered on top.
- Edges: arrow lines between node circles.
- Inventory: shown below each node, icon + number, no multiplication sign.
